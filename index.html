Runbook: Ransomware suspected

  1. Isolate impacted hosts from network.
  2. Disable interactive logons for suspected accounts.
  3. Capture volatile data if safe to do so.
  4. Block IOC domains and hashes at edge and EDR.
  5. Notify incident response lead and legal.
  • Scope affected systems, users, and time window.
  • Collect EDR timeline, Sysmon, Windows Security, firewall, and proxy logs.
  • Contain via segmentation, revoke tokens, reset creds, rotate keys.
  • Eradicate and recover from clean backups; validate integrity.
  • Post-incident review and detection updates.
